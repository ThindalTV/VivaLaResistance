# Google Play Store Listing — Viva La Resistance

## Short Description
Instantly identify resistors with AR. Point your camera, see the value. Works offline. (80 characters)

## Full Description

**Stop squinting at tiny color bands.**

Viva La Resistance uses your phone's camera and on-device machine learning to instantly identify resistor values. Just point your camera at any resistor, and the value appears right there — floating next to the component in real time.

🔬 HOW IT WORKS

The app uses a compact YOLOv8-nano machine learning model running entirely on your device to locate resistors in the camera frame. Color band analysis then calculates the resistance value and tolerance. Results display as clean, readable AR badges anchored to each detected component.

Model size is optimized for mobile — fast inference without draining your battery.

## Assumptions & Notes

1. **Short description** — Exactly 80 characters. Front-loaded with key value propositions: instant, AR, camera, offline.

2. **Full description** — Used visual separators (━━━) and emoji headers for scannability on the Play Store. Android users often skim.

3. **Formatting** — Google Play supports limited HTML in some contexts but markdown-style formatting with Unicode symbols renders well in all views.

4. **Device requirements** — Android 8.0 (API 26) is a reasonable floor for .NET MAUI and covers 95%+ of active devices. Confirm with Shuri.

5. **Feature graphic** — Described rather than created (outside my deliverables as UX Designer). Shuri or an external designer should produce the actual asset. Spec is detailed enough for execution.

6. **Category** — "Tools" is more appropriate for Android (equivalent to iOS "Utilities"). "Education" as secondary captures the student audience.

7. **Content rating** — "Everyone" under IARC/ESRB. No violence, no user-generated content, no purchases, no data collection.

8. **Monetization disclosure** — Google Play requires disclosure of any monetization. The "friendly dialog" framing is accurate and compliant since no IAP exists.
