## Summary

**Status:** ✅ Complete

Three-tier error handling strategy designed and implemented across VivaLaResistance:

1. **Global unhandled exception handlers** in `MauiProgram.cs`:
   - `AppDomain.CurrentDomain.UnhandledException` catches fatal crashes
   - `TaskScheduler.UnobservedTaskException` marks unobserved async exceptions as observed
   - All logged to ILogger; user-facing alert on termination

2. **Graceful degradation in ML pipeline**:
   - `OnnxResistorLocalizationService` model load failures → log, return successfully
   - Inference failures → catch `OnnxRuntimeException`, return empty list
   - Frame pipeline never crashes; "no detection" is safe fallback

3. **Event-based camera error propagation**:
   - Custom exceptions: `CameraPermissionException`, `CameraUnavailableException`
   - `IFrameSource.ErrorOccurred` event for structured error notification
   - UI ViewModels subscribe, display banners; decoupled from platform code
